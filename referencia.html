Guia HTML 5

Tags HTML

Tags Descrição
< var >

Tag utilizada para declarar uma variável na texto ou expressão numérica
< select >

Tag utilizada para criar uma lista drop-down com elementos para serem selecionados.
< mark >

Tag utilizada para marcar parte de um texto.
< details >

Tag utilizada para criar divisões no documento.
< th >

Tag utilizada para criar divisões no documento.
< noscript >

Tag utilizada para definir uma seção de html a ser inserida se um tipo de script não é suportado pela página ou se o script está desativado no navegador.
< bdo >

Tag é usada para substituir a direção do texto atual.
< address >

Tag define as informações de contato do autor/proprietário de um documento ou artigo.
< nav >

Tag é uma seção com links de navegação.
< samp >

Tag é tag frase, texto de exemplo para a definição de programas de computador.
< table >

Define uma tabela.
< input >

Define um controle de entrada.
< link >

Define a relação entre um documento e um recurso externo (mais usado para vincular a folhas de estilo).
< sup >

Define um texto sobrescrito.
< button >

Define um botão clicável.
