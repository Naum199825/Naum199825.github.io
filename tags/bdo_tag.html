Tag HTML <bdo>

Definição e Uso

A tag HTML <bdo>

O <bdo> tag é usado para substituir a direcionalidade atual do texto. Isso faz com que a direcionalidade dos personagens seja ignorada em favor da direcionalidade especificada.

Exemplo

Uso do bdo(Valor = rtl):

<!DOCTYPE html>

<html>

<body>

<p> Este paragrafo irá da Esquerda-para-Direita! </p>

<p> <bdo dir="rtl"> Este paragrafo irá da Direita-para-Esquerda! </bdo> </p>

</body>

</html>

Resultado

Este paragrafo irá da Esquerda-para-Direita!

Atributos

Atributo Valor Descrição
dir ltr Indica que o texto ficará da Esquerda para a Direita (Left-To-Right).
dir rtl Indica que o texto ficará da Direita para a Esquerda (Reft-To-Light).

Atributos Globais

O <bdo> também suporta todos os Atributos Globais do HTML.

Atributos de Eventos

O <bdo> tag também suporta o Evento de Atributos do HTML!

Diferenças entre HTML 4.01 e HTML5

Nenhum!

Configurações CSS Padrão

A maioria dos navegadores irá exibir o <bdo> elemento com os seguintes valores padrão:

bdo{

unicode-bidi: bidi-override;

}
