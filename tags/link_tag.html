Tag HTML <link>

Definição e Uso

A tag <link> define a relação entre o documento atual e um recurso externo.

A tag <link> é mais frequentemente usada para criar links para folhas de estilo externas ou para adicionar um favicon ao seu site.

O elemento <link> é um elemento vazio, contém apenas atributos.

Exemplo

Link para uma folha de estilo externa:

<head>
  <link rel="stylesheet" href="styles.css">
</head>
