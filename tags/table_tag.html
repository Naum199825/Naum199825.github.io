Tag HTML <table>

Definição e Uso

A tag <table> define uma tabela HTML. Uma tabela HTML consiste em um elemento <table> e um ou mais elementos <tr>, <th> e <td>.

O elemento <tr> define uma linha de tabela, o elemento <th> define um cabeçalho de tabela e o elemento <td> define uma célula de tabela.

Uma tabela HTML também pode incluir elementos <caption>, <colgroup>, <thead>, <tfoot> e <tbody>.

Exemplo

Uma tabela HTML simples, contendo duas colunas e duas linhas:

<table>
  <tr>
    <th>Mês</th>
    <th>Economias</th>
  </tr>
  <tr>
    <td>Janeiro</td>
    <td>$100</td>
  </tr>
</table>

Tabela:

Mês Economias
Janeiro $100

Atributos

A tag <table> suporta todos os atributos gerais do HTML.

A tag <table> suporta todos os atributos de evento do HTML.

Outros exemplos

Como criar cabeçalhos de tabela:

<p>Cabeçalho Horizontal:</p>
  <table>
  <tr>
    <th>Nome</th>
    <th>Email</th>
    <th>Telefone</th>
  </tr>
  <tr>
    <td>Lucas Flavio</td>
    <td>lucas@example.com</td>
    <td>123-45-678</td>
  </tr>
</table>
<p>Cabeçalho Vertical:</p>
  <table>
  <tr>
    <th>Nome</th>
    <td>Lucas Flavio</td>
  </tr>
  <tr>
    <th>Email</th>
    <td>lucas@example.com</td>
  </tr>
    <th>Telefone</th>
    <td>123-45-678</td>
  </tr>
</table>

Cabeçalho Horizontal:

Nome Email Telefone
Lucas Flavio lucas@example.com 123-45-678

Cabeçalho Vertical:

Nome: Lucas Flavio
Email: lucas@example.com
Telefone: 123-45-678
