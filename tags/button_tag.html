Tag HTML <button>

Definição e Uso

A tag <table> define um botão clicável.

Dentro de um elemento <button> você pode colocar texto (e tagg como <i>, <b>, <strong>, <br>, <img>, etc.). Isso não é possível com um botão criado com o elemento <input>!

Uma tabela HTML também pode incluir elementos <caption>, <colgroup>, <thead>, <tfoot> e <tbody>.

Dica: Sempre especifique o atributo type para um elemento <button>, para informar aos navegadores que tipo de botão ele é.

Dica: Você pode facilmente estilizar botões com CSS!

Exemplo

Um botão clicável com a cor do background alterada:

<head>
  <style>
    .button1{background-color: #030303;}
  </style>
</head>
<body>
  <button class="button button1"onclick="alert('Hello world!')" >Clique Aqui!</button>
</body>

Botão:

Ao clicar no botão a mensagens Hello World! é retornada.

Atributos

A tag <table> suporta todos os atributos gerais do HTML.

A tag <table> suporta todos os atributos de evento do HTML.
