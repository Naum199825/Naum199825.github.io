Tag HTML <samp>

Definição e Uso

A tag HTML <samp>

O <samp> A tag <samp> é um tag frase. Ele define a saída de amostra a partir de um programa de computador. O conteúdo dentro da <samp></samp> é visualido em fonte monotype Lucida Console.

Dica:

Esta marca não é obsoleto, mas é possível conseguir o efeito mais rica com CSS.

Todas as Tags Frase:

Etiqueta Descrição
<em> Processa como texto enfatizado.
<strong> Define Texto importante.
<code> Define um código de computador.
<samps> Define saída de exemplo a partir de um programa de computador.
<kbd> Define a entrada de teclado.
<var> Define uma variável.

Exemplo

Uso do samp:

<!DOCTYPE html>

<html>

<body>

<samp>

Este texto vai ser visualizado em fonte monotype Lucida Console.

</samp>

</body>

</html>

Atributos Globais

O <samp> também suporta todos os Atributos Globais do HTML.

Atributos de Eventos

O <samp> tag também suporta o Evento de Atributos do HTML!

Configurações CSS Padrão

samp{
