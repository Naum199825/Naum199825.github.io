Tag HTML <sup>

Definição e Uso

A tag <sup> define o texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor. O texto sobrescrito pode ser usado para notas de rodapé, como WWW.

Exemplo

Texto sobrescrito:

<p> Este texto contém um texto <sup>sobrescrito </sup> </p>

Este texto contém um texto sobrescrito

A tag <table> suporta todos os atributos de evento do HTML.

Configurações padrão de CSS

A maioria dos navegadores exibirá o elemento <sup> com os seguintes valores padrão:

sup {
  vertical-align: super;
  font-size: smaller;
}
