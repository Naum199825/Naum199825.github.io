Tag HTML <details>

Definição e Uso

A tag <details> é usada para criar um widget interativo que o usuário pode abrir e fechar. Por padrão, o widget fica fechado. Quando aberto, ele espande e expõe seu conteúdo.

Vários tipos de conteúdos podem ser colocado dentro da tag <details>.

Dica: A tag <summary> é usada em conjunto com <details> para especificar um cabeçalho visível para os detalhes.

Exemplo

Especificando detalhes que o usuário pode abrir e fechar:

<details>

         <summary>Epcot Center </summary>

         <p>

                  O Epcot é um parque temático no Walt Disney World Resort com atrações emocionantes, pavilhões internacionais, fogos de artifício premiados e eventos especiais sazonais.

         </p>

</details>

Saída:

Epcot Center

O Epcot é um parque temático no Walt Disney World Resort com atrações emocionantes, pavilhões internacionais, fogos de artifício premiados e eventos especiais sazonais.

Atributos

A tag <details> suporta todos os atributos gerais do HTML.

A tag <details> suporta todos os atributos de evento do HTML.

Atributo Valor Descrição
open open Especifica se os detalhes já devem vir abertos para o usuário.
