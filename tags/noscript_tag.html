Tag HTML <noscript>

Definição e Uso

A tag HTML <noscript>

O <noscript> tag define um conteúdo alternativo para os usuários que têm roteiros deficientes em seu navegador ou têm um navegador que não suporta script. O <noscript> elemento pode ser usado tanto em <head> e <body> . Quando usado dentro do <head> elemento: <noscript> deve conter apenas <link>, <style> e <meta> elementos. O conteúdo dentro do <noscript> elemento será exibido se os scripts não são suportados, ou estão desativados no navegador do usuário.

Dica: Dica: Também é uma boa prática para usar o comment tag para scripts de "esconder" a partir de browsers sem suporte para scripts do lado do cliente (para que eles não mostre-os como texto simples):

<noscript>

<!--

function displayMsg() {

alert("Hello World!")

}

// -->

<noscript>

Exemplo

Uso do noscript:

<noscript>

document.write("Hello World!")

</noscript>

<noscript> Seu navegador não suporta JavaScript! </noscript>

Atributos Globais

O <noscript> também suporta todos os Atributos Globais do HTML.

Diferença entre HTML e XHTML

Em XHTML, o <noscript> tag não é suportado.

Diferenças entre HTML 4.01 e HTML5

Em HTML 4.01, o <noscript> tag só pode ser usado dentro do <body> elemento.

Em HTML5, o <noscript> tag poder ser usado tanto dentro do <head> e <body>.
