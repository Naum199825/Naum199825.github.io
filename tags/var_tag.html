Tag HTML <var>

Definição e Uso

A tag <var> é usada para definir uma variável em programação ou expressões matemáticas. O seu conteúdo é tipicamente mostrado em itálico.

Dica: Esta tag não está obsoleta. Porém, é possível obter um efeito mais rico usando CSS.

Exemplo

Definindo partes de um texto como variáveis em um documento:

<p> A área de um trinângulo é: 1/2 * <var>b</var> * <var>h</var>, onde <var>b</var> é o tamanho da base e <var>h</var> é a altura. </p>

Saída:

A área de um trinângulo é: 1/2 * b * h, onde b é o tamanho da base e h é a altura.

Atributos

A tag <var> suporta todos os atributos gerais do HTML.

A tag <var> suporta todos os atributos de evento do HTML.

Outros exemplos

Definindo uma expressão de soma de variáveis:

<p> Soma: <var>x</var> + <var>y</var> = <var>z</var>. </p>

Saída:

Soma: x + y = z.
