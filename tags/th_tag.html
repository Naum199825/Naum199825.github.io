Tag HTML <th>

Definição e Uso

A tag <th> define uma célula de cabeçalho de uma tabela HTML.

Uma tabela HTML tem dois tipos de células:

  •          > Células de cabeçalho - que contém as informações do cabeçalho (criada com a tag <th>).

  •          > Células de dados - que contém os dados da tabela (criada com a tag <td>).

Os elemento em <th> são em negrito e centralizados por padrão.

Os elementos em <td> são normais e alinhados à esqueda por padrão.

Exemplo

Criando uma tabela simples com três linhas, duas colunas e quatro células de dados:

<table>

         <tr>

                  <th> Mês </th>

                  <th> Economias </th>

         </tr>

         <tr>

                  <td> Janeiro </td>

                  <td> $100 </td>

         </tr>

         <tr>

                  <td> Fevereiro </td>

                  <td> $90 </td>

         </tr>

</table>

Saída:

Mês Economias
Janeiro $100
Fevereiro $90

Atributos

A tag <var> suporta todos os atributos gerais do HTML.

A tag <select> suporta todos os atributos de evento do HTML.

Atributo Valor Descrição
abbr text Especifica uma versão abreviada do conteúdo da célula.
colspan number Especifica o número de colunas que uma célula de cabeçalho deve abranger.
headers header_id Especifica uma ou mais células de cabeçalho às quais uma célula está relacionada.
rowspan number Especifica o número de linhas que uma célula de cabeçalho deve abranger.
scope

col

colgroup

row

rowgroup

 Especifica se uma célula de cabeçalho é um cabeçalho para uma coluna, linha ou grupo de colunas ou linhas.
