Tag HTML <select>

Definição e Uso

A tag <select> é usada para criar uma lista de elementos que podem ser selecionados por um usuário.

O atributo "name" é necessário para referenciar o dado selecionado depois que o elemento for selecionado. (Caso você omita o atributo "name", nenhum dado será coletado quando um elemento da lista for selecionado.)

O atributo "id" é necessário para identificar a lista com uma legenda.

A tag <option> dentro da tag <select> serve para indicar um elemento da lista que pode ser selecionado.

Dica: Sempre adicione a tag <label> para melhor acessibilidade.

Exemplo

Criando uma lista com quatro opções:

<label for="carros">Escolha um carro: </label>

<select name="carros" id="carros" >

         <option value="volvo"> Volvo </option>

         <option value="saab"> Saab </option>

         <option value="mercedes"> Mercedes </option>

         <option value="audi"> Audi </option>

</select>

Atributos

A tag <select> suporta todos os atributos gerais do HTML.

A tag <select> suporta todos os atributos de evento do HTML.

Atributo Valor Descrição
autofocus autofocus Especifica que aquela lista deve ter foco automatico quando a página carregar.
disabled disabled Especifica que aquela lista deve ser desabilitada.
form form_id Define a qual formulário aquela lista pertence.
multiple multiple Especifica que multiplas opções podem ser selecionadas de uma vez.
name name Define o nome da lista drop-down.
required required Especifica que é obrigatória a seleção de um valor antes de enviar o formulário.
size number Define o número de elementos visíveis da lista.

Outros exemplos

Usando <select> com tag <optgroup> :

<label for="carros">Escolha um carro: </label>

<select name="carros" id="carros" >

         <optgroup label="Carros Suecos" >

                 <option value="volvo"> Volvo </option>

                 <option value="saab"> Saab </option>

         </optgroup>

         <optgroup label="Carros Alemães" >

                 <option value="mercedes"> Mercedes </option>

                 <option value="audi"> Audi </option>

         </optgroup>

</select>
