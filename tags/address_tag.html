Tag HTML <address>

Definição e Uso

A tag HTML <address>

O <address> tag define as informações de contato do autor/proprietário de um documento ou artigo. As informações de contato podem ser um endereço de e-mail, URL, endereço físico, número de telefone, identificador de mídia social, etc. O texto no elemento <address> geralmente é renderizado em itálico, e os navegadores sempre adicionam uma quebra de linha antes e depois do elemento <address>.

Exemplo

Uso do address:

<address>

Escrito por <a href="mailto:ethanwolfsvein234@gmail.com"> Ethan Loyola </a>. <br>

Nos Visite em: <br>

Exemplo.com. <br>

Estrada do Lobo 25, Salinas-MG <br>

Brasil

</address>

Atributos Globais

O <address> também suporta todos os Atributos Globais do HTML.

Atributos de Eventos

O <address> tag também suporta o Evento de Atributos do HTML!

Configurações CSS Padrão

A maioria dos navegadores irá exibir o <address> elemento com os seguintes valores padrão:

address{

display: block;

font-style: italic;

}
